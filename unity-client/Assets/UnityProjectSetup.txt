Unity Project Setup Instructions for PvP Game
==============================================

1. Unity Version: 2022.3 LTS or later
2. Target Platform: Android
3. Render Pipeline: Universal Render Pipeline (URP)

Setup Steps:
-----------

1. Create new Unity project with URP template
2. Import required packages:
   - Input System (com.unity.inputsystem)
   - Socket.IO for Unity (com.github.socketio.socket-io-client)
   - Newtonsoft JSON (com.unity.nuget.newtonsoft-json)
   - TextMeshPro (com.unity.textmeshpro)
   - Addressables (com.unity.addressables)

3. Set up scenes:
   - LoginScene.unity
   - LobbyScene.unity  
   - GameScene.unity
   - ResultScene.unity

4. Configure Player Settings for Android:
   - Minimum API Level: Android 9 (API level 28)
   - Target API Level: Latest
   - Scripting Backend: IL2CPP
   - Architecture: ARM64
   - Graphics APIs: OpenGLES3

5. Set up URP:
   - Create URP Asset
   - Configure for mobile performance
   - Set up rendering layers

6. Import scripts from Assets/Scripts directory
7. Set up prefabs in Assets/Prefabs
8. Configure Addressables groups
9. Build and test on Android device

Required Dependencies:
--------------------
- Socket.IO Client for Unity (GitHub package)
- Newtonsoft JSON for Unity
- Unity Input System
- TextMeshPro
- Addressables System

Performance Targets:
------------------
- Target FPS: 60
- Max snapshot processing time: <5ms
- Max input sending time: <2ms
- Zero GC allocations in game loop
- Build size: <100MB

Scene Flow:
---------
LoginScene -> LobbyScene -> GameScene -> ResultScene -> LobbyScene

Network Configuration:
--------------------
- Server URL: http://localhost:3000 (configurable)
- WebSocket Namespace: /pvp
- JWT Authentication via handshake
- Automatic reconnection: 5 attempts, 1s delay
- Ping timeout: 30s
- Ping interval: 25s

Input Configuration:
------------------
- Touch: Movement via touch drag, Fire via tap, Ability via swipe
- Keyboard (dev): WASD movement, Space to fire, E for ability
- Input rate: 60 FPS
- Timestamp precision: milliseconds

Game Configuration:
-----------------
- Tick rate: 20Hz (50ms per tick)
- Player speed: 5 units/sec (0.25 units per tick)
- Map size: 100x100 units
- Max health: 100 HP
- Fire range: 10 units, 10 damage
- Ability range: 15 units, 25 damage, 5s cooldown
- Shield duration: 5s (100 ticks)

Object Pooling:
-------------
- Bullets: 50 pre-warmed
- Effects: 20 pre-warmed
- Zero instantiation during gameplay

Addressables Setup:
-----------------
- Ships: PlayerShip.prefab, EnemyShip.prefab
- Bullets: Bullet.prefab
- Effects: Explosion.prefab, Shield.prefab, MuzzleFlash.prefab
- UI: MatchUI.prefab, ResultScreen.prefab
- Scenes: All scenes as Addressables

Error Handling:
-------------
- Network timeout: Reconnect flow
- Server disconnect: Return to lobby
- Invalid token: Re-login
- Match state mismatch: Force disconnect
- High latency warning: >200ms

Debug Features:
-------------
- FPS counter
- Ping display
- Position validation
- Anti-cheat warnings
- Input visualization
